# Defender Status Review

## Purpose

Document Windows Security / Microsoft Defender status as endpoint security evidence.

## Evidence Captured

- Virus & threat protection status
- Protection updates status
- Current threats status
- Virus and threat protection settings status
- Ransomware protection status as visible in Windows Security
