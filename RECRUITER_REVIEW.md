## What A Recruiter Should Notice

This project shows a practical endpoint investigation workflow from an IT support foundation. It is useful for teams that need someone who can validate endpoint posture, collect evidence, review logs, document timelines, and escalate clearly.

## Proof Points

- Reviewed Microsoft Defender / Windows Security posture.
- Used PowerShell output and Event Viewer evidence.
- Built a basic endpoint investigation timeline.
- Documented containment and remediation recommendations.
- Kept scope honest and avoided advanced EDR or malware-analysis claims.
